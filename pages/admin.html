📋 How to connect HTML files to your Worker

Open each HTML file and find this line near the top of the <script> section:

const WORKER = 'REPLACE_WITH_YOUR_WORKER_URL';

Replace it with your actual Worker URL shown above. Do this in: index.html, student-dashboard.html, homework.html, teacher-dashboard.html.
